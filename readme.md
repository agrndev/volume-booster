# Volume Booster
A Mozilla add-on that boosts your tab volume beyond 100%.

## Installation
- Download the `volume-booster.xpi` compressed file on the `releases` section of this repository
- Open Mozilla/Zen browser and write `about:addons` on the URL
- Click on the settings icon > Install add-on from file
- Select the `volume-booster.xpi` file

## Features
- Boost volume up to 600%
- Quick preset buttons (50%, 100%, 200%, 300%, MAX)
- Visual volume indicator
- Per-tab volume control
- Remembers your settings

## Permissions
- `activeTab` - Access the current tab's audio
- `storage` - Save your preferences
- `tabs` - Manage tab audio
